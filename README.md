# :construction_worker: How work ?
```bash
# Clone this repository
$ git clone https://github.com/KaikySantos/dtmoney.git
# Access the project folder
$ cd dtmoney
# Install the dependencies
$ yarn install
# Run the application
$ yarn start
```
Go to http://localhost:3000/ to see the result.
